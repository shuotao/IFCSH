### Notes:

The IFC-SG Toolkit
The Toolkit is the foundation of your data structure. Ensure the project team has downloaded and applied the following:
Setting Parameters: Defining the required IFC-SG shared parameters within the BIM environment.
Validation: Using the toolkit to check if parameter values meet the regulatory requirements.
Excel Mapping: Utilizing the IFC-SG mapping files to ensure Revit Categories align correctly with IFC Classes.

### Notes:

<!-- Slide number: 5 -->
5 Steps to Prepare an IFC-SG Model
To ensure a smooth submission, follow this linear workflow:
Add IFC-SG Parameters: Use the toolkit to inject required fields into the model.
Populate IFC Classes: Assign correct IFC Entities (e.g., IfcWall, IfcBeam) to elements.
Populate IFC-SG Values: Fill in the project-specific data required by the agencies.
Validate Parameters: Check for missing or incorrectly formatted data.
Export to IFC: Generate the .ifc file using the specific IFC-SG export settings.

### Notes:

<!-- Slide number: 6 -->
Submission Good Practices
1. Geospatial & Level Alignment
Shared Coordinates: Models must be geo-referenced (SVY21) to ensure perfect overlay in the CORENET X portal.
Vertical Consistency: M&E levels must match Architectural levels exactly. Use Copy/Monitor to avoid elevation discrepancies.
2. Model Performance & Optimization
The 800 MB Rule: Individual file sizes must not exceed 800 MB.
Model Splitting: If a model exceeds the limit, split by System (e.g., MVAC, Plumb, Fire) or Zone to maintain processing speed.

### Notes:

<!-- Slide number: 7 -->
Submission Good Practices
3. Modeling Integrity
Correct Revit Categorization: Use the Suggested Revit Representation. Avoid "Generic Models"—assign equipment to the correct Family Category to ensure accurate IFC-SG entity mapping.
Spatial Coordination: Models must be fully coordinated and clash-cleared against other disciplines prior to submission.
4. Quality Assurance (QA) Procedures
Schedule-Based Audits: Use Revit Schedules to cross-check that all elements have been tagged with mandatory IFC-SG parameters.
Validation Check: If a parameter is blank in the schedule, it will fail the automated compliance check.
